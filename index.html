SISTEMAS DE NUMERAÇÃO

Introdução

Um numeral é um símbolo ou grupo de símbolos que representa um número. Dependendo da escrita, época ou sistema de numeração que é usado, um mesmo numeral pode indicar números diferentes. Um sistema de numeração é uma forma de representar números a partir de numerais de uma forma consistente. Por exemplo, dependendo do sistema de numeração utilizado, o numeral "11" pode ser interpretado como o numeral romano para dois, o numeral binário para três ou o numeral decimal para onze.
Os sistemas de numeração baseiam-se no posicionamento de cada algarismo, servindo para representar as unidades, dezenas, centenas, e assim por diante, da direita para a esquerda. Cada uma dessas, por sua vez, representa uma potência do sistema em questão. No sistema decimal, por exemplo, sistema mais comum atualmente, representam as potências de 10, tendo 10^0 para as unidades, 10^1 para as dezenas, e assim em diante. Já o sistema binário é baseado em potências de 2, representadas pelos algarismos 0 e 1 apenas.

Sistema binário

Como dito anteriormente, o sistema binário baseia-se em potências de 2 e é representado apenas por 2 algarismos: 0 e 1. É um dos sistemas mais comuns na computação por conta de sua simplicidade e representatividade "metafórica": o 0 pode ser interpretado como negativo, desligado ou apagado, e o 1 pode indicar um sistema positivo, ligado ou aceso, por exemplo. A desvantagem desse sistema é que, por utilizar apenas 2 algarismos para cada potência, os números no sistema binário podem ficar muito mais longos do que em sistemas de bases maiores.

Sistema hexadecimal

Ao contrário do sistema binário, que tem como característica principal sua simplicidade, o sistema hexadecimal tem foco em criar números menores, consumindo menos bits para uma mesma informação, se comparado ao sistema binário e até mesmo ao decimal. Ele se baseia em potências de 16, utilizando 16 algarismos para representar seus números: além dos usuais algarismos de 0 a 9, utilizados no sistema decimal, o sistema hexadecimal utiliza, ainda, as letras A, B, C, D, E e F para representar os números de 10 a 15. O número 14, por exemplo, seria escrito no sistema hexadecimal apenas como E.
